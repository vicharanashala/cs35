# CS35 - Crowdsource FAQ Platform

##  Project Title
Crowdsource FAQ Generation Platform



##  Project Description
This project is a web-based FAQ platform designed to allow users to view, ask, and answer frequently asked questions.

It is a crowdsourced system where users can contribute by adding new questions and providing answers.

The platform is designed with:
- Flexibility
- Easy usability for students
- Interactive learning support



##  Pages in the Project

1. FAQ General Page
   - Displays commonly asked questions and answers

2. Ask New FAQ Page
   - Users can submit new questions

3. Answer FAQ Page
   - Users can provide answers to existing questions



## Project Structure
- frontend → User interface (HTML/CSS/JS or framework)
- backend → Server logic and API handling
- faqData.json → Stores FAQ data


## Features
- View FAQs
- Submit new questions
- Answer existing questions
- Easy navigation between pages
